¿Qué es la web?

La web es un sistema de documentos interconectados a través de enlaces, accesibles a través de Internet. Fue inventada por Tim Berners-Lee en 1989 y ha revolucionado la forma en que compartimos información y nos comunicamos. La web se basa en tecnologías como HTML, CSS y JavaScript para crear páginas web interactivas y dinámicas.

Elementos importantes para la creación de una página web

  • HTML: Estructura del contenido.
  • CSS: Estilo y diseño visual.
  • JavaScript: Interactividad y dinamismo.
